히포 교육 컷툰 자동생성기

줄거리와 대사를 입력하면 컷별 장면 구성, 대사 가이드, 이미지 생성 프롬프트를 자동으로 만들어주는 버전입니다.
캐릭터 설정은 고정되어 있으며 모든 컷에서 동일하게 유지됩니다.

입력 설정

주인공 갈색 머리 / 남색 정장 + 흰 셔츠 + 넥타이 / 신입·직원 느낌 / 앉아서 듣거나 필기하는 역할

히포 파란색·하늘색 로봇형 마스코트 / 동그란 얼굴 / 귀처럼 보이는 헤드셋·안테나 / 귀엽고 친절한 표정
※ 이 캐릭터 설정은 프롬프트에 자동으로 고정 반영됩니다.
여러 줄로 나눠 적으면 컷별 장면 분배가 자연스러워집니다.
이름: 대사 형식으로 입력하면 컷별로 자동 배정됩니다.
캐릭터 고정 컷별 프롬프트 자동 생성 복사 기능 보완

※ 현재 버전은 실제 이미지를 직접 생성하지 않고,
컷 구성과 이미지 생성용 프롬프트를 자동 생성하는 1차 버전입니다.
